# PolyRedeemer

PolyRedeemer is a script designed to automatically check and redeem positions. The script runs periodically and interacts with the blockchain using the credentials provided in your configuration file.

---

## How To Use It

It is very easy to install and run the script. Please follow the steps below:

1.  **Download the script:** Download the `polyredeemer.js` script.
2.  **File location:** Place the script in the **SAME** folder where you keep your `.env` file.
3.  **Configuration:** Ensure you have added your keys to your `.env` file as follows:
    * `PK=0x...` (Your private key)
    * `PBK=0x...` (Your public key)

## Running the Script

The script is set to check for positions every 15 minutes.

* **Server:** It is recommended to set up a script on your server to let it run 24/7.
* **Manual:** If you do not have a server, you can also run the script manually whenever you want.
